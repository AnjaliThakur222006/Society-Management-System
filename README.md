# Society Management System

A web-based Society Management System developed using Python Flask.

## Features
- Resident Management
- Visitor Entry System
- Maintenance Bills
- Event Management
- Complaints System
- Security Dashboard

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- MySQL

## Project Structure
app.py – Main Flask application  
templates/ – HTML pages  
static/ – CSS, JS, images  
tests/ – Testing files  

## How to Run
1. Install Python
2. Install Flask
3. Run:

python app.py

Then open localhost in browser.
